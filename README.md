## Features

 * A simple to use, flexible query builder for elastic search.

## Requirements

 * php: ~7.0
 * nikic/php-parser: ~3.0

## Installation

Through [Composer](http://getcomposer.org) as [saxulum/saxulum-elasticsearch-querybuilder][1].
